Repositório dedicado aos meus estudos e exercícios práticos em **Análise de Dados**. Aqui você encontrará projetos e desafios que explorei para desenvolver habilidades em manipulação, visualização e modelagem de dados usando Python e bibliotecas do ecossistema de dados.

## Objetivo

Documentar minha jornada de aprendizado em Análise de Dados, aplicando conceitos estatísticos e de machine learning a conjuntos de dados reais. Este repositório serve tanto como portfólio pessoal quanto como material de referência para outros estudantes.

## Estrutura do Repositório

O repositório está organizado em notebooks Jupyter, cada um abordando um tema ou projeto específico:

### Estudos de Machine Learning
| Arquivo | Descrição |
| :--- | :--- |
| `Machine_learning_TITANIC.ipynb` | Projeto clássico de classificação para prever sobreviventes do Titanic. |
| `Random_Forest_Animal_Table_01.ipynb` | Introdução ao algoritmo Random Forest para classificação de espécies animais. |
| `Random_Forest_Animal_Table_02.ipynb` | Continuação do estudo com Random Forest, refinando o modelo. |

### Estudos de Regressão e Análise Policial
| Arquivo | Descrição |
| :--- | :--- |
| `Data_Analyses_Estudo_de_caso_da_policia_de_Seattle.ipynb` | Análise exploratória de dados de paradas policiais em Seattle. |
| `Regression_Study_Boston_Police.ipynb` | Aplicação de modelos de regressão para dados da polícia de Boston. |

### Fundamentos e Estudos Binários
| Arquivo | Descrição |
| :--- | :--- |
| `Binery_Study_Bank_01.ipynb` a `Binery_Study_Bank_06.ipynb` | Série de notebooks sobre conceitos fundamentais de análise binária e preparação de dados para classificação. |

### Documentação
| Arquivo | Descrição |
| :--- | :--- |
| `Readme.txt` | Versão inicial da documentação do projeto. |

## Tecnologias e Bibliotecas Utilizadas

- **Linguagem:** Python 3.x
- **Ambiente de desenvolvimento:** Jupyter Notebook / Google Colaboratory
- **Principais bibliotecas:**
  - `pandas` para manipulação de dados.
  - `scikit-learn` para modelos de machine learning.
  - `matplotlib` e `seaborn` para visualizações.
  - Provável uso de `numpy` e `plotly` em alguns notebooks.

## Como Explorar Este Repositório

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/0xFAB10/Data-Analysis-Exercises.git
2.  **Navegue pelos notebooks:** Eles estão numerados e nomeados de forma autoexplicativa. Recomendo começar pelos estudos binários (Binery_Study_Bank_01 ao 06) para entender os fundamentos.

3.  **Execute localmente:** Você precisará de um ambiente Jupyter. Instale as dependências necessárias conforme for encontrando erros de importação (sugiro um ambiente virtual com pip install jupyter pandas scikit-learn matplotlib seaborn).
